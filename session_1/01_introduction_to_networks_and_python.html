Introduction to Python and Jupyter Notebooks

Python is a very popular high-level programming language. If you have never programmed before, you can think of programming as using a very powerful calculator, one that works not only with numbers but with any kind of object a computer can represent. These objects include numbers, of course, but also words or characters (called strings), lists, matrices, functions, dictionaries, and even custom-made structures such as networks.

Python runs code by reading a sequence of instructions written in a text file known as a script. In this workshop, however, we will primarily work with Jupyter Notebooks, which offer a more interactive and beginner-friendly environment.

A Jupyter Notebook is an interactive workspace where you can run code, write notes, and explore datasimilar to notebooks in RStudio. A notebook is divided into cells. Some cells contain Python code that you can execute independently, while others contain text written in Markdown (like the cell you are reading now). This setup makes notebooks ideal for learning, experimenting, and documenting your work all in one place.

Like any calculator, Pythons basic functionality can be extended through packages (also called libraries). These are add-ons written by others to provide specific features that are not built into core Python. For example, the package matplotlib allows us to create plots and visualize data. Unlike app-store plugins, Python packages are free and widely shared within the community. In this first tutorial, we will cover the basics of programming in Python and introduce a few essential packages for data and network analysis:

  • NumPy for working with numerical data and arrays
  • Pandas for reading, saving and handling datasets in table form
  • Matplotlib for plotting and visualizing data
  • NetworkX the most widely used Python library for analyzing networks

By the end, youll have a foundation strong enough to dive into network analysis and the more advanced tools used throughout the workshop.

Outline of the tutorial

  1. Simple operations: Learn the basic python operations, starting from using it as a calculator to handling different types
  2. Functions: Learn how to use built-in functions as well to define your own
  3. Control flow: Learn the basics of if-else blocks and for loops.
  4. Importing libraries: Learn to import python libraries/packages for extra functionalities
  5. Basic network analysis with networkx: Import library networkx to construct a network and perform a basic analysis.

Jupyter notebook basics

Before we begin, here are a few Jupyter Notebook basics that will help you move smoothly through the tutorial.

A Jupyter Notebook is organized into cells, and each cell can be either:

  • a Code cell (where you write and run Python code), or
  • a Markdown cell (where you write formatted text, notes, or explanations).

You can switch a cells type using the toolbar at the top, or with keyboard shortcuts:

  • Esc M to turn a cell into Markdown
  • Esc Y to turn it into Code

To run a cell, press:

  • Shift + Enter runs the cell and moves to the next one
  • Ctrl + Enter runs the cell but stays in place

You can add new cells from the toolbar or using:

  • Esc A to insert a cell above
  • Esc B to insert a cell below
  • Esc C to copy a selected cell
  • Esc X to cut a selected cell
  • Esc V to paste the copied cell below

To delete a cell, press:

  • Esc D D (press D twice)

A few helpful tips:

  • Jupyter keeps track of the order in which you run cells. If something behaves oddly, you can use the Kernel Restart option to reset the environment and run your cells again in order.
  • You can see a cells output immediately below itgreat for experimenting interactively.
  • Markdown cells let you create titles, lists, equations, code snippets, and more, making notebooks excellent for combining analysis and explanation.

These simple commands will make the notebook environment feel much more intuitive as we go along.

3. Control Flow

When writing code, we often want our program to make decisions or repeat certain actions. This is where control flow comes in. Control flow tools allow Python to decide what to do next based on conditions or to repeat tasks automatically. Two of the most common tools are if statements and for loops.

if statements: making decisions

An if statement lets your code run only when a certain condition is true.
You can think of it as setting rules:

If something happens, then do this. Otherwise, do something else.

The usual Python if-statement block looks something like

if condition:
    # do something
else:
    # Do something else
In[22]:
# For example, let's use an if statement to check the temperature and print a message accordingly.
temperature = 32

if temperature > 30:
    print("It's hot outside!")
else:
    print("The temperature is fine.")


# The statement `temperature > 30` is a Boolean (True or False), which evaluates whether the if condition is met.
temperature > 30, type(temperature > 30)

It's hot outside!
Out[22]:
(True, bool)

for loops: repeating actions

A for loop lets you repeat the same action for each element in a sequence (like a list, string, or range of numbers). Think of it as telling Python:

Repeat this operation for every item in this collection.

The usual Python for-loop block looks something like

for element in sequence:
    # do something with `element`
In[23]:
# For example, let's use a for loop to iterate over a list of fruits and print a message for each fruit.

# Do some operation for each fruit in the list

    print(f"I like {fruit}.")

# We can also get the index of each fruit using the enumerate() function
for i, fruit in enumerate(fruits):
    print(f"Fruit {i}: {fruit}")

def consider_as_friend(friendships, person): # define function
    """Return all people who say that `person` is their friend. """ # add docstring
    
    person_friends = []  # initialize list of people who consider `person` a friend
    
    for other in friendships: # for loop
        if person in friendships[other]: # if statement

4. Importing libraries

As we said before, libraries are like plugins that extend the functionality of core Python.

There are two main ways to import libraries. The first way looks something like

import numpy as np

Here, we're importing library numpy and will call it with the shorthand name np. We can use np to methods from numpy.

The second way involves importing specific methods from a library, something like

from numpy import mean

Here, we import the mean method from numpy, which we can call as an ordinary function.

4.1. NumPy

NumPy is one of the core libraries in the Python scientific ecosystem. It is very fast and designed for working with numerical data, especially lists, arrays, matrices, and general linear algebra tasks.

At first, NumPy may feel a bit confusing because it introduces its own data structure called the NumPy array (numpy.ndarray). Although it looks similar to the regular Python lists we saw earlier, it behaves differently and is optimized for mathematical operations. Once you get used to it, NumPy becomes an essential tool for any data or network analysis workflow.

Here are the most important things to know when starting with NumPy:

  • NumPy arrays behave like mathematical vectors.
    You can add, subtract, multiply, or apply functions to them element-wise with very simple syntax.

  • You can easily create arrays from Python lists, ranges, or even automatically using functions like np.zeros(), np.ones(), or np.arange().

  • NumPy provides fast linear algebra tools, such as matrix multiplication, dot products, transposes, and more.

  • NumPy arrays support slicing and indexing, which let you extract specific portions of your data (similar to lists, but more powerful).

4.2. Pandas

Pandas is one of the most important Python libraries for working with data. It allows you to load, explore, clean, and manipulate datasets in a way that feels similar to working with Excel or R data frames, but with the power and flexibility of Python.

The main structure in pandas is the DataFrame, which you can think of as a table: rows represent observations, and columns represent variables. Pandas makes it extremely easy to:

  • convert arrays (from numpy or Python lists) into DataFrame objects
  • load data from CSV, Excel, or other formats
  • explore and summarize variables
  • select and filter rows
  • create or transform new columns
  • handle missing data
  • group and aggregate information

If we want to convert existing data (say, from a list or matrix X) into a DataFrame, we can use

  pd.DataFrame(X)

The DataFrame method has a lot of optional arguments (such as naming the columns however you want), so I recommend checking its documentation (using ?pd.Dataframe).

If we want to read data from a file, we can use the pd.read_* method (where * will be the format of the file to be read). For instance, for reading a CSV file, we would do

  pd.read_csv('path_to_file.csv')

Again, I recommend checking ?pd.read_csv, as there several optional arguments.

5. Network analysis with networkx

In Python, the most widely used library for network analysis is NetworkX. It provides an intuitive interface for creating, manipulating, analyzing and visualizing networks. Graphs in networkx are of a special type Graph (for undirected networks) or DiGraph (for directed networks).

NetworkX lets us construct the same network in several ways, depending on the type of data we have. For instance

  1. From an adjacency matrix
    (for example, stored as a NumPy array)

  2. From an edgelist
    (for example, a pandas DataFrame with source and target columns)

  3. Directly from a list of edges
    (a Python list of tuples)

Once we have a network, we can apply several methods to analyze them. A few common methods in NetworkX are the following:

Concept / Measure Description NetworkX Function
Node degree Number of neighbors a node has G.degree(n) or dict(G.degree())
Network density Fraction of possible edges that actually exist nx.density(G)
Average clustering coefficient Mean clustering across all nodes nx.average_clustering(G)
Degree assortativity Correlation between degrees of connected nodes nx.degree_assortativity_coefficient(G)
Shortest path (single pair) Length of shortest path between two nodes nx.shortest_path_length(G, u, v)
Average shortest path length Mean distance between all pairs of nodes nx.average_shortest_path_length(G)
Diameter Longest shortest path in the network nx.diameter(G)
Connected components Groups of nodes that are mutually reachable nx.connected_components(G)
Number of connected components Total number of components nx.number_connected_components(G)

Example: Let's consider the following very simple network with 5 nodes:

One way of representing this network is as a tuple $G = (V,E)$ of vertices $V$ and edges $E$, such that:

  • Nodes: $V = [1, 2, 3, 4, 5]$
  • Edges: $E = [(1, 2), (1, 3), (2, 3), (3, 4), (3, 5), (4, 5)]$

We can also represent it as an edgelist (where pandas DataFrames will come useful).

source target

1 3
2 3
3 4

4 5

Finally, we can represent it as an adjacency matrix (where numpy arrays will come useful).

1 2 3 4 5
1 0 1 1 0 0
2 1 0 1 0 0
3 1 1 0 1 1
4 0 0 1 0 1
5 0 0 1 1 0

Obs: This adjacency matrix is symmetric because the network is undirected.
